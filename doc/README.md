# dart_vault Documentation

Документация пакета `dart_vault` — универсального Data Layer для AQ экосистемы.

## 📚 Структура документации

### 🏗️ Архитектура (`architecture/`)
Фундаментальные архитектурные документы:
- `ARCHITECTURE.md` — полное описание архитектуры системы
- `LOGGED_STORABLE_CONVENTION.md` — конвенции для LoggedStorable

### 📖 Руководства (`guides/`)
Практические руководства по использованию:
- `USAGE_GUIDE.md` — полное руководство пользователя
- `QUICK_START.md` — быстрый старт для новых пользователей
- `migration_plan_v2.md` — план миграции на PostgreSQL

### 📊 Отчёты (`reports/`)
Отчёты о разработке и аудитах:
- `COMPLIANCE_REPORT.md` — отчёт о соответствии production требованиям
- `PRODUCTION_READY_STATUS.md` — статус готовности к production

### 🗄️ Архив (`archive/`)
Исторические документы и отчёты о завершённых этапах:
- Отчёты о сессиях разработки
- Отчёты о завершённых этапах
- Устаревшие планы и статусы

## 🚀 С чего начать?

1. **Новичкам**: начните с `guides/QUICK_START.md`
2. **Разработчикам**: изучите `architecture/ARCHITECTURE.md`
3. **DevOps**: смотрите `guides/USAGE_GUIDE.md` для deployment

## 📄 Корневые документы

В корне пакета остаются только:
- `README.md` — основной README пакета
- `CHANGELOG.md` — история изменений версий
